# chatgpt

This repository includes a simple Connect 4 game in Python.

## Running the game

Run the interactive game with:

```bash
python3 'connect 4/connect4.py'
```

## Running the tests

A small test script ensures the core game logic works:

```bash
python3 'connect 4/test_connect4.py'
```

## Web version

Run the Flask backend to try the web-based game:

```bash
python3 'connect 4/server.py'
```

Then open [http://localhost:5000](http://localhost:5000) in your browser.
